# TangZhiZzz's Portfolio

我的个人主页 | My Personal Portfolio

🔗 访问地址: https://tangzhiizz.github.io

## 技术栈

- HTML5
- CSS3
- Vanilla JavaScript

## 特点

- ✨ 响应式设计，支持移动端
- 🎨 渐变背景和动画效果
- 🌙 深色主题
- 📱 移动端优化的导航菜单

## 本地运行

```bash
# 直接用浏览器打开 index.html
open index.html

# 或使用简单的 HTTP 服务器
python3 -m http.server 8000
```

## 自定义

1. 修改 `index.html` 中的个人信息
2. 更换头像图片（在 index.html 中修改头像链接）
3. 更新社交链接
4. 修改技能和简介内容
